## 下載各種套件:
<b> 到終端機 (Win+R -> 輸入cmd) 輸入以下程式: </b>
- pip3 install torch torchvision
- pip install matplotlib
- pip install numpy
- pip install notebook

> 載入各種套件:
> -  torch
> -  torchvision
> - matplotlib
> - numpy
> - notebook (指jupyter notebook)

## 開啟 Jupyter Notebook

**在終端機直接輸入:**
- jupyter notebook
