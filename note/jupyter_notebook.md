# 關於 Jupyter Notebook 的使用方法
## 下載各種套件:
<b> 到終端機輸入以下程式: </b>
- pip3 install torch torchvision
- pip install matplotlib
- pip install numpy
- pip install notebook

> 載入各種套件:
> -  torch
> -  torchvision
> - matplotlib
> - numpy
> - notebook (指jupyter notebook)

## 開啟 Jupyter Notebook

**在終端機輸入:**
- jupyter notebook
